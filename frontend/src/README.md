# Frontend Source Layout

Struttura iniziale consigliata:

- `app/` bootstrap client, router UI, provider
- `pages/` entry page e schermate top-level
- `features/` workflow e moduli applicativi
- `components/` componenti riusabili
- `api/` client HTTP e typing contratti
- `lib/` utilita pure frontend
- `styles/` entry CSS e tema
